# ui-to-web

**从 UI 设计到网站上线 · 单文件交互教程网页**

一个自包含的单文件 HTML 网页（CSS / JS 全部内联，零外部依赖），系统讲解从 UI 设计、代码实现到静态部署上线的完整流程。

## 内容章节
- 01 总览路径
- 02 UI 设计（Google Stitch 工作流）
- 03 代码实现（HTML / CSS / JS 三件套）
- 04 GitHub（建仓库与推送）
- 05 部署方案（GitHub Pages / Cloudflare Pages / Vercel / Netlify）
- 06 资源库（国内外优质模板）
- 07 上线检查清单

## 本地预览
直接用浏览器打开 `index.html` 即可，无需任何构建步骤。
